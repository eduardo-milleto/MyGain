# Project Structure

```
frontend/
  docs/
  guidelines/
  src/
    app/
      components/
      contexts/
    styles/
  index.html
  package.json
  postcss.config.mjs
  vite.config.ts
```

## Conventions

- UI text must be English.
- Component filenames use `PascalCase`.
- Hook and util filenames use `camelCase`.
- Keep shared components in `src/app/components`.
- Put global styles and tokens in `src/styles`.
